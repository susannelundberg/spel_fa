# Fiskarens Arv

**Fiskarens Arv** är ett textbaserat fantasyäventyr under **aktiv utveckling**.  
Spelet är inte komplett och nya uppdrag, fiender och artefakter kommer att läggas till i framtiden.

## Hur man spelar
1. Starta spelet.
2. Välj namn och vapen för din karaktär.
3. Utforska byn eller skogen och möt fiender.
4. Slå tärning för att avgöra strider.
5. Samla delar av **Fiskens Hjärtas Amulett**.

## Status
- 🟡 Under utveckling  
- Vissa uppdrag och fiender saknas fortfarande.  
- Sparfunktion fungerar för nuvarande karaktär.  

## Framtida idéer
- Fler uppdrag och fiender.  
- Magiska bonusar från artefakten.  
- Slutboss.
